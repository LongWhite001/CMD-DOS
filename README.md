# Abaixo oque cada opção faz: 
(ATENÇÃO: Por Padrão as Versões ONLINE Estão sempre atualizadas, as versões OFFLINE não possuem
atualização, desde sua implementação.)

(1)- Powershell Security Desabled:
Desabilita a Segurança do Windows Powershell. ele lhe pedira um numero digite 4
caso não funcione tente outro numero. o numero pode mudar dependendo
da versão do windows.

(2)- Ativar Windows e Office ONLINE:
Massgrave sera Carregado e executado. Precisa de Internet.

(3)- Brute Force Attack on Zip Files:
Ataque de força bruta em arquivos .zip, .7z, RAR.
ele usara a wordlist presente na pasta. esta wordlist é
personalizavel a seu gosto, assim podendo adicionar mais senhas nela.
ele mostrara a senha se encontrada em uma janela POPUP.

(4)- Ativar Windows OFFLINE KMS.
Este modo tem varias chaves de ativação para Windows 10 e 11 offline.
possui também chaves para o windows 8 e 8.1. caso nem uma chave ative, tente o 
Random Mode.
Em algumas versões do Windowsa chave do Windows 10 vale para ativar o Windows 11.

(5)- Ativar Office  OFFLINE:
Ohook. não precisa de internet, ele ativara o pacote office offline.

(6)- WinUtil ONLINE:
Ele carregara o Winutil ONLINE (Precisa de Internet).
o Winutil possibilita varias alterações no windows e até mesmo
instalação de programas.

(7)- WinUtil OFFLINE:
Versão OFFLINE do Winutil. (Não Precisa de Internet.)

(8)- Corrigir Painel de Visualização do Windows:
Corrigir erro do windows Defender que bloqueia a visualização de PDFs
no Painel Visualizar Lateral.

(9)- Menu Esteganografia:
Permite esconder arquivos como txt, planilhas, arquivos de word, e virus, .bat, 
e outros até certo tamanho dentro de uma imagem. Ele também permite remover depois se 
quiser o arquivo. A imagem continua abrindo normalmente, é uma otima opção para 
esconder arquivos sigilosos.
